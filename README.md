# Customer Churn & Revenue Risk Analytics Platform

## Overview

Developed an end-to-end analytics solution to analyse customer churn behaviour and quantify revenue at risk using Python, SQL, Streamlit and Plotly.

The project identifies key churn drivers, analyses customer segments, and provides business recommendations to improve retention and reduce revenue loss.

## Technologies

* Python
* SQL
* Streamlit
* Plotly
* Pandas
* Git & GitHub

## Key Features

* Interactive Streamlit dashboard
* Customer churn analysis
* Revenue-at-risk calculation
* Customer segmentation
* KPI tracking
* Business recommendations

## Business Questions Answered

* What is the current customer churn rate?
* Which customer groups are most likely to churn?
* How much revenue is at risk?
* Which retention strategies should be prioritised?

## Dashboard Metrics

* Total Customers
* Churn Rate
* Revenue
* Revenue at Risk

## Outcome

Identified high-risk customer segments and provided actionable recommendations focused on retention, contract optimisation and customer lifetime value improvement.
